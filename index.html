Speech Samples

The model is evaluated with bilingual speakers from:
the CMU Arctic Database

We have 7 different voice conversion systems with different configurations. They are compared in three groups:
(1) Using different features for Voice Conversion:
In this case study, there 4 systems for comparison:
(a) Mel_VC : The baseline voice conversion system, where Mel-spectrogram features are used as the voice conversion network input.
(b) S3R_VC : The baseline voice conversion system that only takes S3R representation as the input features to the voice conversion network.
(c) Mel-S3R_VC : The voice conversion system with the proposed Mel-S3R speech representation as the input features to the voice conversion network.

(2) Using different self-attention blocks:
The second group presents an ablation study about the self-attention. We compare the following 4 systems:
(a) FUSE : In this system, the Mel-S3R network only consists of the fuse-level fuse-attention block (FFA) while the input and output attention blocks are discarded.
(b) ISA-FFA : The Mel-S3R network has the input attention block input-level self-attent (ISA) and the FFA. Features obtained by such a Mel-S3R network are used as the voice conversion network input.
(c) FFA-OSA : In this system, the speech representation is obtained by the FFA and the output-level self-attention (OSA) blocks.
(d) Mel-S3R_VC : The proposed Mel-S3R network with ISA, FFA, and OSA.

(3) Different combination methods:
The third group focuses on the ablation study about feature combination methods by comparing 3 systems:
(a) CONCAT: The Mel-spectrogram and S3R features are concatenated and used as input features for the voice conversion network.
(b) FUSE: The same system in the second comparison group (2)(a), where only the fuse-attention is used to combine the features.
(c) Mel-S3R_VC : The proposed Mel-S3R features which are obtained by the 3 attention blocks in the combination network.

Source: Speech from a source speaker, we hope to preserve the speech content.
Target: Speech from a target speaker, we hope the converted speech sound like this speaker's voice.
