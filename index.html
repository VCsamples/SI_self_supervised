There are two systems implemented for comparison:
(1) Baseline: This is the PPG-based XVC framework. It is the baseline system without the linguistic cycle consistency loss.
(2) CyclePPG-XVC: This is proposed XVC framework with linguistic cycle consistency loss.

Source: Speech from a source speaker, we hope to preserve the speech content.
Target: Speech from a target speaker, we hope the converted speech sound like this speaker's voice.
Reference: Reference speech from the target speaker speaking the source content.
